# TIA Portal V17 Shortcuts

These shortcuts were transcribed from user-provided TIA Portal V17 screenshots. Use them when the user asks for V17 shortcuts or when UI automation by keyboard is more reliable than menu navigation.

## General

- Save project: `Ctrl+S`
- Compile object: `Ctrl+B`
- Open project: `Ctrl+O`
- Close project: `Ctrl+W`
- Print: `Ctrl+P`
- Undo: `Ctrl+Z`
- Redo: `Ctrl+Y`
- Find next: `F3`
- Confirm: `Ctrl+Q`
- Refresh: `F5`
- Select all: `Ctrl+A`
- Rename object: `F2`
- Delete object: `Del`
- Delete item: `Ctrl+E`
- Go to line: `Ctrl+G`
- Repeat last operation: `Ctrl+Y`

## Copy Paste And Insert

- Copy object: `Ctrl+Ins`
- Copy object: `Ctrl+C`
- Cut object: `Shift+Del`
- Cut object: `Ctrl+X`
- Paste grid contents: `Ctrl+V`
- Paste clipboard contents: `Shift+Ins`
- Insert row below: `Alt+Ins`
- Insert newline in an input field or insert a table row above: `Ctrl+Enter`

## Editor And Window Switching

- Previous editor: `Alt+Ctrl+Left`
- Previous editor: `Ctrl+Shift+F6`
- Next open editor: `Alt+Ctrl+Right`
- Next open editor: `Ctrl+F6`
- Close active editor: `Ctrl+F4`
- Close all open editors: `Ctrl+Shift+F4`
- Close code section: `Ctrl+Shift+Num-`
- Open code section: `Ctrl+Shift+Num+`
- Close all code sections: `Ctrl+Shift+Divide`
- Open all code sections: `Ctrl+Shift+Multiply`
- Horizontal split editor space: `Ctrl+F12`
- Vertical split editor space: `F12`
- Remove window split: `Alt+Shift+F12`
- Reduce editor view size: `Ctrl+Num-`
- Increase editor view size: `Ctrl+Num+`
- Set editor view to 100%: `Ctrl+0`

## Navigation

- Go to next workspace in TIA Portal: `F6`
- Go to previous workspace: `Shift+F6`
- Up browse: `Shift+Escape`
- Up browse: `Alt+Up`
- Down browse: `Alt+Down`
- Browse to local toolbar: `Alt+F10`
- Switch between project view and Portal view: `Alt+F7`
- Switch between project view and library view: `Alt+F5`
- Open this cross-reference: `F11`
- Show cross-reference in the inspector window: `Shift+F11`
- Open block or PLC data type: `F7`
- Toggle bookmark: `Ctrl+Shift+M`
- Previous bookmark: `Ctrl+Shift+5`
- Next bookmark: `Ctrl+Shift+6`
- Toggle mark: `Ctrl+OemPeriod`
- Go offline: `Ctrl+M`
- Go online: `Ctrl+K`

## CPU And Download

- Download to device: `Ctrl+L`
- Start CPU: `Ctrl+Shift+E`
- Stop CPU: `Ctrl+Shift+Q`
- Start runtime system: `Ctrl+Shift+Z`
- Start simulation: `Ctrl+Shift+X`

## Layout And Views

- Restore active window layout: `Shift+Alt+0`
- Use window layout 1: `Shift+Alt+1`
- Use window layout 2: `Shift+Alt+2`
- Use window layout 3: `Shift+Alt+3`
- Use window layout 4: `Shift+Alt+4`
- Use window layout 5: `Shift+Alt+5`
- Open or close details view: `Ctrl+4`
- Open or close inspector window: `Ctrl+5`
- Open or close overview window: `Ctrl+2`
- Open inspector tab Properties: `Ctrl+6`
- Open inspector tab Info: `Ctrl+7`
- Open inspector tab Diagnostics: `Ctrl+8`
- Show or hide task cards: `Ctrl+3`
- Show or hide project tree: `Ctrl+1`
- Show accessible devices: `Ctrl+U`
- Show program status: `Ctrl+T`
- Show object browser: `Ctrl+J`
- Show find and replace task card: `Ctrl+H`
- Show auto-complete list: `Ctrl+I`
- Show screen keyboard: `Ctrl+Shift+K`
- Show or hide the Ctrl+9 panel from the screenshot: `Ctrl+9`

## Library And Project Commands

- Open library: `Ctrl+Shift+L`
- Open HWCN online editor: `Ctrl+D`
- Project save as: `Ctrl+Shift+S`
- Open Services and Support in TIA Portal: `Ctrl+Shift+7`
- Close application: `Alt+F4`
